Welcome

Below are some of the projects I've worked on:

Valgen

A Barret Model values diagram generator.

A default set of values can be selected which correspond to a level on the diagram and a table value. Custom values can be added which are stored locally for later use.

Values in the diagram tables are colour coded depending on the relationship between personal, current and desired values.

Build with Electron, React, React Redux and Material UI. Written mostly in Typescript.

Pixy Java Camera

This is part of my final year project at University.

Using Java I modelled the behaviour of a Pixy Camera and a location controller.

More detail on this project is available on GitHub and upon request.

Trakontrol

This is an Android application that can be used to control Traktor (DJ Software) via mobile phone.

The computer running Traktor would host a server using Node.js which the mobile device could connect to when on the same network allowing MIDI control.

IMDB Show Data Scraper

A simple web scraper to get show ratings from IMDB.

Built with JavaScript, uses Cheerio (JQuery) to get specific data from IMDB pages.

Graph TV

A recreation of a web app I used in the past called GraphTV which has gone offline.

Uses the IMDB web scraper tool to display season rating trends for TV shows. React front-end, Express for API.

Vale House Farm

A website built using Wordpress for a local venison provider.

I've provided them with an easy way to update and maintain their website via the Wordpress dashboard by editing pages on the built-in editor.

Code provided is the theme which was based on the Understrap theme.

Upsall Castle Sporting

A basic website using Bootstrap.

This website was mainly for contact, displaying dates and a photo gallery. The gallery is connected to the customers' Flickr.

Code provided shows HTML markup & other code for the website
